# GuroFlow PH

AI-Powered Curriculum-Aligned Teaching Platform for Filipino Educators

## Vision

To provide teachers with high-quality curriculum-aligned instructional materials powered by AI.

## Features

- AI DLL Generator
- AI Lesson Builder
- Quiz Generator
- Worksheet Generator
- PowerPoint Generator
- DOCX/PDF Export
- Curriculum Library
- Teacher Dashboard
- School Dashboard

## Tech Stack

- Next.js
- NestJS
- PostgreSQL
- Prisma
- Tailwind CSS
- OpenAI API
