# 🎓 GuroFlow DLL PRO

AI-Powered Curriculum-Aligned Teaching Platform for Filipino Educators

---

## 📖 Overview

GuroFlow DLL PRO is a modern web application that helps Filipino teachers create, customize, and download curriculum-aligned Daily Lesson Logs (DLL), worksheets, quizzes, assessments, and instructional materials.

---

## ✨ Features

- 📚 DLL Library
- 🤖 AI Lesson Builder
- 📝 Quiz Generator
- 📄 Worksheet Generator
- 📊 Rubrics Generator
- 📑 DOCX Export
- 📕 PDF Export
- 👩‍🏫 Teacher Dashboard
- 🏫 School Dashboard
